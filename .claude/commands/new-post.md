# /new-post — 新記事の下書きを作成する

## 目的
新しいブログ記事の下書きファイルを `content/drafts/` に生成する。

## 実行手順

1. ユーザーに以下を確認する（未指定の場合のみ）：
   - 記事のテーマ・ネタ（何について書くか）
   - カテゴリ（design / photo / book / film / note）
   - タイトル案（なければ提案する）

2. 以下のフォーマットで下書きファイルを作成する：
   - パス: `content/drafts/YYYY-MM-DD_slug.md`
   - 日付は今日の日付（JST）
   - スラッグは英語・ハイフン区切り

3. ファイルの内容：
   - Frontmatterを `.claude/rules/frontmatter.md` の仕様に従って記述
   - `draft: true` を必ずセット
   - 本文は「導入（2〜3文）」「本題のプレースホルダー」「締めのプレースホルダー」の骨格だけ用意する
   - プレースホルダーは `<!-- ここに書く -->` 形式

4. 作成後、ファイルパスとタイトルを報告する

## 禁止事項
- 本文を完成させない（骨格だけ作る）
- ユーザーの指示なく `content/posts/` に置かない

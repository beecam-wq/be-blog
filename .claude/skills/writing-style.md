# Writing Style Skill — Yasuhide Ura のブランドボイス

## 文体の特徴（参考例から抽出）
- 短い文と長い文が交互に来るリズム感
- 観察から入り、感情・考察へと展開する構成
- 「〜なんだろうか」「〜かもしれない」の問いかけを多用
- 比較や対比で特徴を浮き彫りにする手法

## 使いやすい語彙・表現
- 「ふと気づいた」「なんとなく」「どこかに」（自然な気づきの表現）
- 「〜という感じ」「〜みたいな」（断定しない柔らかさ）
- 「おもしろい」「いい」（シンプルな評価語を素直に使う）

## 避けるべき表現
- 「〜することが重要です」（説教調）
- 「〜について解説します」（ハウツー感）
- 「まとめると」「結論として」（論文調）
- 「ぜひ」「必ず」（押しつけ感）
